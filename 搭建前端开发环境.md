Visual Studio Code (简称 VS Code / VSC) 是一款免费开源的现代化轻量级代码编辑器，支持几乎所有主流的开发语言的语法高亮、智能代码补全、自定义快捷键、括号匹配和颜色区分、代码片段、代码对比 Diff、Git 命令等特性，支持插件扩展，并针对网页开发和云端应用开发做了优化。软件跨平台支持 Windows、Mac 以及 Linux，运行流畅，可谓是微软的良心之作。
